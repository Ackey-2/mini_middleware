# Phase 4 Shared-Memory Zero-Copy Implementation Plan

> **For agentic workers:** TDD, one commit per task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** When discovery reports a same-host match, stream published messages between two processes through a lock-free shared-memory ring instead of TCP; cross-host keeps the Phase 3 TCP path.

**Architecture:** A machine-level `host_id` is advertised in discovery. `DataPlane` compares it against `MatchInfo.remote_host_id`: equal → SHM, else → TCP. On a same-host match the publisher creates a per-`(publisher,topic)` POSIX shared-memory segment holding a lock-free SPMC ring and registers a `ShmSink` (an `ISink`) in `LocalBus`; the subscriber opens the same segment (deterministic name) and a `ShmReaderManager` poller drains it into `LocalBus::deliver_inbound`. Both sides act for SHM (unlike TCP where only the publisher dials).

**Tech Stack:** C++17, POSIX shm (`librt`), lock-free `std::atomic`, Protobuf, GoogleTest, CMake.

**Spec:** [docs/superpowers/specs/2026-06-18-phase4-shm-design.md](../specs/2026-06-18-phase4-shm-design.md)
